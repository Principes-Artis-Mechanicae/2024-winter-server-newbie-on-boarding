# 점프_투_자바_2장

클래스

- 메서드
    - 명령문

String을 사용할 수 있음.

클래스 이름 규칙

- 클래스명은 명사로 한다.
- 클래스명은 대문자로 시작한다.
- 여러 개의 단어를 조합하여 만들 경우 각 단어의 첫 번째 글자는 대문자이어야 한다. (파스칼 케이스(pascal case))

메서드 이름 규칙

- 메서드명은 동사로 한다.
- 매서드명은 소문자로 시작한다.
- 여러 개의 단어를 조합하여 만들 경우 맨 첫 글자를 제외한 나머지 단어의 첫 글자는 대문자를 써서 구분한다. (카멜 케이스(camel case))
